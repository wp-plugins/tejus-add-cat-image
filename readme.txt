Upload `dj_cat_image` to the `/wp-content/plugins/` directory1. Activate the plugin through the 'Plugins' menu in WordPress1. Place `<?php tejus_dj_getcatimg($catid) ?>` in your templates== Frequently Asked Questions === what will this function return? =this function will return url of the associated image.= what it will return if category dont have any associated image =it will return 0.== Screenshots ==1.
